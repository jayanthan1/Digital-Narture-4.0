Overlaying Badges on Images with Positioning Utilities

About This Exercise

This exercise demonstrates how to use Bootstrap's positioning utilities to overlay badges on images:

  • position-relative - Applied to the container element
  • position-absolute - Applied to the badge element
  • Position utilities like top-0, start-0, etc. - Control badge placement

These utilities allow you to place badges, labels, or other elements precisely over images or other content.

How Relative and Absolute Positioning Work Together

The Positioning Relationship:
  1. The parent element is set to position-relative
  2. The badge/overlay is set to position-absolute
  3. The badge is positioned using utilities like top-0, end-0, etc.
  4. The badge is now positioned relative to its parent, not the page
<div class="position-relative">
    <img src="..." class="img-fluid" alt="...">
    <span class="position-absolute top-0 start-0 badge bg-primary">
        New
    </span>
</div>
Key Positioning Utilities:
  • top-0, top-50, top-100 - Vertical position (top, middle, bottom)
  • start-0, start-50, start-100 - Horizontal position from start (left in LTR)
  • end-0, end-50, end-100 - Horizontal position from end (right in LTR)
  • translate-middle - Centers an element when used with top-50 start-50
  • translate-middle-x - Centers horizontally only
  • translate-middle-y - Centers vertically only

Centered Badge Positioning

You can center badges using the translate-middle utilities:

Center
position-absolute top-50 start-50 translate-middle
Center Top
position-absolute top-0 start-50 translate-middle-x
Center Left
position-absolute top-50 start-0 translate-middle-y

Badge with Offset

You can use margin utilities to offset badges from their positioned corners:

Hot Deal hot deal
position-absolute top-0 end-0 translate-middle
The translate-middle class shifts the badge to be centered on the corner rather than inside it.
New new item
position-absolute top-0 start-100 translate-middle
Using start-100 positions the badge at the right edge, and translate-middle centers it on that point.
