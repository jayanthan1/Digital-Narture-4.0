Bootstrap Container, Row, and Column Classes

This example demonstrates the proper use of .container, .row, and .col-* classes for responsive design.

1. Container Types

Fixed-Width Container

The .container class provides a responsive fixed-width container with different max-widths at each breakpoint.

<div class="container"> Content here </div>

Fluid Container

The .container-fluid class provides a full-width container that spans the entire width of the viewport.

<div class="container-fluid"> Content here </div>

Breakpoint-Specific Containers

Bootstrap also provides breakpoint-specific containers that are 100% wide until the specified breakpoint.

<div class="container-sm"> <div class="container-md"> <div class="container-lg"> <div class="container-xl"> <div class="container-xxl">

Container Max-Widths

  • .container has different max-widths for each breakpoint:
  • Small (≥576px): 540px
  • Medium (≥768px): 720px
  • Large (≥992px): 960px
  • X-Large (≥1200px): 1140px
  • XX-Large (≥1400px): 1320px

2. Row Class

Basic Row

The .row class creates a horizontal group of columns with proper gutters. It must be placed within a container.

<div class="container"> <div class="row"> <!-- Columns go here --> </div> </div>

Row with No Gutters

Use .g-0 to remove gutters between columns.

<div class="row g-0"> <!-- Columns with no gutters --> </div>

3. Column Classes

Basic Column Classes

Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content. It's built with flexbox and is fully responsive.

<div class="col-12">Full width</div> <div class="col-6">Half width</div> <div class="col-4">One-third width</div> <div class="col">Auto width</div>

Responsive Column Classes

Use breakpoint-specific column classes for responsive layouts.

<div class="col-12 col-md-6 col-lg-4"> <!-- Full width on mobile, half on medium, one-third on large screens --> </div>

4. Putting it All Together

Best Practices

  1. Always start with a .container or .container-fluid
  2. Add .row as a direct child of the container
  3. Place .col-* classes as direct children of the row
  4. Content should go inside the columns
  5. Use responsive column classes to control layout across different screen sizes
<div class="container"> <div class="row"> <div class="col-12 col-md-6 col-lg-4"> <!-- Content here --> </div> <div class="col-12 col-md-6 col-lg-4"> <!-- Content here --> </div> <div class="col-12 col-md-6 col-lg-4"> <!-- Content here --> </div> </div> </div>

Common Mistakes to Avoid

  • Placing a .row directly in the <body> without a container
  • Nesting a .container inside another .container
  • Placing content directly in a .row without a column
  • Using column classes that add up to more than 12 in a single row
