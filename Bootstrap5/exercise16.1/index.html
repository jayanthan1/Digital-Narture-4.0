Fixed Footer with Bootstrap Positioning Utilities

About This Exercise

This exercise demonstrates how to create a fixed footer using Bootstrap's positioning utilities:

  • position-fixed - Positions an element relative to the viewport
  • bottom-0 - Positions the element at the bottom of its container
  • w-100 - Makes the element take up 100% width

The footer will remain visible at the bottom of the screen even as you scroll through the page content.

You can use the buttons in the bottom-right corner to toggle between different footer styles.

How Fixed Positioning Works

When an element is set to position-fixed, it is removed from the normal document flow and positioned relative to the viewport. This means it will stay in the same position even when the page is scrolled.

Key Utilities Used:
  • position-fixed - Fixes the element relative to the viewport
  • bottom-0 - Positions the element at the bottom (0px from bottom)
  • start-0 or end-0 - Positions horizontally (0px from left or right)
  • w-100 - Makes the element take up 100% width
Important Considerations:
  • Fixed elements are removed from the normal document flow
  • Content can be hidden behind fixed elements
  • Add padding to body to prevent content from being hidden
  • Consider z-index for proper stacking of fixed elements
<footer class="position-fixed bottom-0 w-100">
    <!-- Footer content here -->
</footer>

Basic Fixed Footer Example

Below is the code for a basic fixed footer that stays at the bottom of the viewport:

<footer class="position-fixed bottom-0 start-0 w-100 bg-light border-top p-3">
    <div class="container">
        <div class="d-flex justify-content-between align-items-center">
            <span>© 2025 Company Name</span>
            <ul class="list-inline mb-0">
                <li class="list-inline-item"><a href="#">Privacy</a></li>
                <li class="list-inline-item"><a href="#">Terms</a></li>
                <li class="list-inline-item"><a href="#">Support</a></li>
            </ul>
        </div>
    </div>
</footer>

This creates a footer that:

  • Stays fixed at the bottom of the viewport (position-fixed bottom-0)
  • Spans the full width of the viewport (w-100)
  • Has a light background with a top border
  • Contains copyright information and links

Footer Variations

Fixed footers can be styled in various ways to match your website's design. Here are some common variations:

Basic Footer

A simple footer with light background and minimal content.

class="position-fixed bottom-0 start-0 w-100 bg-light"
Dark Footer

A dark-themed footer with white text for contrast.

class="position-fixed bottom-0 start-0 w-100 bg-dark text-white"
Colorful Footer

A footer with a colorful background using Bootstrap's contextual colors.

class="position-fixed bottom-0 start-0 w-100 bg-primary text-white"

Common Use Cases

Fixed footers are useful in various scenarios:

Cookie Consent Banners

A fixed banner at the bottom of the screen to inform users about cookie policies:

<div class="position-fixed bottom-0 start-0 w-100 bg-light p-3 border-top">
    <div class="container d-flex justify-content-between align-items-center">
        <p class="mb-0">This website uses cookies to ensure you get the best experience.</p>
        <button class="btn btn-primary">Accept</button>
    </div>
</div>
Mobile App-like Navigation

A fixed bottom navigation bar for mobile-friendly websites:

<nav class="position-fixed bottom-0 start-0 w-100 bg-white border-top py-2">
    <div class="d-flex justify-content-around">
        <a href="#" class="text-center">
            <i class="bi bi-house"></i>
            <div>Home</div>
        </a>

Best Practices

When using fixed footers, keep these best practices in mind:

Add Body Padding

Add padding to the bottom of the body element to prevent content from being hidden behind the fixed footer:

body {
    padding-bottom: 70px; /* Height of your footer */
}
Consider Mobile Viewports

Ensure your fixed footer works well on mobile devices by keeping it compact and responsive:

@media (max-width: 576px) {
    .fixed-footer {
        padding: 0.5rem 0;
    }
    body {
        padding-bottom: 50px; /* Smaller padding for mobile */
    }
}
Use z-index Appropriately

Set an appropriate z-index to ensure the footer appears above other content:

.fixed-footer {
    z-index: 1030; /* Same as Bootstrap's navbar */
}
Accessibility Considerations

Ensure your fixed footer is accessible to all users:

  • Provide sufficient color contrast
  • Ensure links are keyboard navigable
  • Consider adding a "skip to content" link
  • Test with screen readers
