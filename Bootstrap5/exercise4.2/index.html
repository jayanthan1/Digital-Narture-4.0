How It Works

This layout uses Bootstrap's grid system to create a responsive four-column layout with equal widths:

  • On mobile devices (less than 576px wide), the columns stack vertically
  • On small devices and larger (576px and wider), each column takes 25% width (col-sm-3)
<div class="container"> <div class="row"> <div class="col-sm-3"> Column 1 content here... </div> <div class="col-sm-3"> Column 2 content here... </div> <div class="col-sm-3"> Column 3 content here... </div> <div class="col-sm-3"> Column 4 content here... </div> </div> </div>
Understanding the Grid System

Bootstrap's grid system is based on a 12-column layout. When using col-sm-3, each column takes up 3 of the 12 available columns, which is 25% of the width (3/12 = 1/4 = 25%).

Since 3 Ã— 4 = 12, we can fit exactly 4 columns of equal width in a single row.

Alternative Column Layouts

Two-Column Layout
<div class="row"> <div class="col-sm-6">Column 1</div> <div class="col-sm-6">Column 2</div> </div>

Each column takes 50% width (6/12) on small devices and larger.

Three-Column Layout
<div class="row"> <div class="col-sm-4">Column 1</div> <div class="col-sm-4">Column 2</div> <div class="col-sm-4">Column 3</div> </div>

Each column takes 33.33% width (4/12) on small devices and larger.

Six-Column Layout
<div class="row"> <div class="col-sm-2">Column 1</div> <div class="col-sm-2">Column 2</div> <div class="col-sm-2">Column 3</div> <div class="col-sm-2">Column 4</div> <div class="col-sm-2">Column 5</div> <div class="col-sm-2">Column 6</div> </div>

Each column takes 16.67% width (2/12) on small devices and larger.

Mixed Column Layout
<div class="row"> <div class="col-sm-6">Column 1</div> <div class="col-sm-3">Column 2</div> <div class="col-sm-3">Column 3</div> </div>

First column takes 50% width, the other two take 25% each.
