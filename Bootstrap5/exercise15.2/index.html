Bootstrap Shadows and Rounded Pills

About This Exercise

This exercise demonstrates how to use Bootstrap's shadow and rounded-pill utilities with cards. You'll see examples of:

  • Different shadow sizes (shadow-sm, shadow, shadow-lg)
  • Rounded pill shapes with rounded-pill
  • Combining these utilities with cards for enhanced visual effects

Each example includes the code used to create it and a description of the effect.

Shadow Utilities

Bootstrap provides utilities for adding shadows of different sizes to elements.
