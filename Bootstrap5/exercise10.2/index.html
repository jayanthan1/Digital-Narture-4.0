Tabbed Navigation - Exercise 10.2

This exercise demonstrates how to create tabbed navigation using Bootstrap's nav, nav-tabs, and nav-pills components.

Click on different tabs to see the content change dynamically.

Basic Nav

  • Home
  • Features
  • Pricing
  • Disabled 
<ul class="nav">
    <li class="nav-item">
        <a class="nav-link active" aria-current="page" href="#">Home</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">Features</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">Pricing</a>
    </li>
    <li class="nav-item">
        <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Disabled</a>
    </li>
</ul>

Nav Tabs

Home Tab Content

This is the content for the home tab. It's displayed by default when the page loads.

Tab content can include any HTML elements, including text, images, forms, and more.

Profile Tab Content

This is the content for the profile tab. It's displayed when you click on the Profile tab.

User profile information would go here.

Contact Tab Content

This is the content for the contact tab. It's displayed when you click on the Contact tab.

<!-- Nav tabs -->
<ul class="nav nav-tabs" id="myTab" role="tablist">
    <li class="nav-item" role="presentation">
        <button class="nav-link active" id="home-tab" data-bs-toggle="tab" 
                data-bs-target="#home" type="button" role="tab" 
                aria-controls="home" aria-selected="true">Home</button>
    </li>
    <li class="nav-item" role="presentation">
        <button class="nav-link" id="profile-tab" data-bs-toggle="tab" 
                data-bs-target="#profile" type="button" role="tab" 
                aria-controls="profile" aria-selected="false">Profile</button>
    </li>
    <li class="nav-item" role="presentation">
        <button class="nav-link" id="contact-tab" data-bs-toggle="tab" 
                data-bs-target="#contact" type="button" role="tab" 
                aria-controls="contact" aria-selected="false">Contact</button>
    </li>
</ul>

<!-- Tab content -->
<div class="tab-content" id="myTabContent">
    <div class="tab-pane fade show active" id="home" role="tabpanel" aria-labelledby="home-tab">
        <h4>Home Tab Content</h4>
        <p>This is the content for the home tab.</p>
    </div>
    <div class="tab-pane fade" id="profile" role="tabpanel" aria-labelledby="profile-tab">
        <h4>Profile Tab Content</h4>
        <p>This is the content for the profile tab.</p>
    </div>
    <div class="tab-pane fade" id="contact" role="tabpanel" aria-labelledby="contact-tab">
        <h4>Contact Tab Content</h4>
        <p>This is the content for the contact tab.</p>
    </div>
</div>

Nav Pills

Home Pill Content

This is the content for the home pill. Notice how the pills have a different styling compared to tabs.

Pills have a more button-like appearance with rounded corners and a filled background when active.

Profile Pill Content

This is the content for the profile pill. It's displayed when you click on the Profile pill.

Pills are great for interfaces where you want a more prominent navigation.

Contact Pill Content

This is the content for the contact pill. It's displayed when you click on the Contact pill.
