Bootstrap Form Components Used

1. Basic Form Controls

Bootstrap provides several form control styles, input types, and additional components like labels.

<label for="firstName" class="form-label">First Name</label> <input type="text" class="form-control" id="firstName" required> <div class="invalid-feedback"> Please enter your first name. </div>

Key classes:

  • form-label: Styles the label for form controls
  • form-control: Styles text inputs and textareas
  • invalid-feedback: Provides validation feedback messages

2. Input Groups

Input groups extend form controls by adding text, buttons, or button groups on either side of inputs.

<div class="input-group"> <span class="input-group-text"> <i class="bi bi-envelope"></i> </span> <input type="email" class="form-control" id="email" required> <div class="invalid-feedback"> Please enter a valid email address. </div> </div>

Key classes:

  • input-group: Container for input and addons
  • input-group-text: Styles text, icons, or other content to be displayed alongside inputs

3. Select Menus

Custom select menus with support for multiple options, sizing, and validation states.

<label for="country" class="form-label">Country</label> <select class="form-select" id="country"> <option selected disabled value="">Choose...</option> <option value="us">United States</option> <option value="ca">Canada</option> <!-- More options --> </select>

Key classes:

  • form-select: Styles the select dropdown

4. Checkboxes and Radios

Custom checkbox and radio controls with support for validation states.

<div class="form-check"> <input class="form-check-input" type="checkbox" id="termsCheck" required> <label class="form-check-label" for="termsCheck"> I agree to the Terms and Conditions </label> <div class="invalid-feedback"> You must agree before submitting. </div> </div>

Key classes:

  • form-check: Container for checkboxes and radios
  • form-check-input: Styles the checkbox or radio input
  • form-check-label: Styles the label for the checkbox or radio

5. Form Layout

Bootstrap's grid system can be used within forms to create responsive layouts.

<div class="row mb-3"> <div class="col-md-6"> <!-- Form control for first name --> </div> <div class="col-md-6"> <!-- Form control for last name --> </div> </div>

This creates a two-column layout on medium screens and larger, while stacking the fields on smaller screens.

6. Form Validation

Bootstrap provides styles for validation feedback. For client-side validation, you can use HTML5 form validation attributes and Bootstrap's validation classes.

<!-- Add the novalidate attribute to the form --> <form id="registrationForm" novalidate> <!-- Form controls with required attribute --> </form> <!-- JavaScript for form validation --> <script> // Example starter JavaScript for disabling form submissions if there are invalid fields (function () { 'use strict' // Fetch all the forms we want to apply custom Bootstrap validation styles to var forms = document.querySelectorAll('.needs-validation') // Loop over them and prevent submission Array.prototype.slice.call(forms) .forEach(function (form) { form.addEventListener('submit', function (event) { if (!form.checkValidity()) { event.preventDefault() event.stopPropagation() } form.classList.add('was-validated') }, false) }) })() </script>
