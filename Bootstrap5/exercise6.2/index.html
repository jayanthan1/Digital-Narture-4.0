Card Layout with Flexbox Utilities

This example demonstrates how to use justify-content-between and align-items-center in card layouts.

1. Card Header with Flexbox

This example shows how to create a card header with content justified between left and right, and vertically centered.

Product Name
New

This is a simple card with a header that uses flexbox utilities to place the title on the left and a badge on the right.

<div class="card-header d-flex justify-content-between align-items-center"> <h5 class="card-title">Product Name</h5> <span class="badge bg-primary">New</span> </div>
Key Flexbox Classes Used:
  • d-flex: Creates a flexbox container
  • justify-content-between: Places items at opposite ends of the container
  • align-items-center: Centers items vertically

Card Layout Strategy

In this product grid example, we're using several flexbox techniques:

  • Each card body uses d-flex flex-column to stack content vertically
  • The price and button container uses justify-content-between align-items-center
  • We use flex-grow-1 on the description to push the price/button area to the bottom
  • All cards have the same height using h-100 class
