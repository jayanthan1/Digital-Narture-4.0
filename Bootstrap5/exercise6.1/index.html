Responsive Navbar with Flexbox Utilities

This example demonstrates how to create a responsive navbar using Bootstrap's flexbox utilities like d-flex, flex-column, and flex-md-row.

1. Basic Responsive Navbar

This navbar stacks vertically on mobile (less than 768px) and horizontally on medium screens and larger (768px and up).

Brand Logo
Home About Services Portfolio Contact
<div class="d-flex flex-column flex-md-row align-items-center"> <div class="logo mb-3 mb-md-0">Brand Logo</div> <div class="d-flex flex-column flex-md-row"> <a href="#" class="nav-link active">Home</a> <a href="#" class="nav-link">About</a> <a href="#" class="nav-link">Services</a> <a href="#" class="nav-link">Portfolio</a> <a href="#" class="nav-link">Contact</a> </div> </div>
Key Flexbox Classes Used:
  • d-flex: Creates a flexbox container
  • flex-column: Stacks flex items vertically
  • flex-md-row: Arranges flex items horizontally on medium screens and larger
  • align-items-center: Centers items vertically

2. Navbar with Justified Content

This navbar uses justify-content-between to place the logo on the left and navigation links on the right.

The example above uses Bootstrap's navbar component, which automatically handles the responsive behavior and includes a toggle button for mobile view.

6. Understanding Flexbox Utilities in Bootstrap

Key Flexbox Utilities

Utility Class Description CSS Equivalent
d-flex Creates a flexbox container display: flex;
d-inline-flex Creates an inline flexbox container display: inline-flex;
flex-row Sets the flex direction to row (horizontal) flex-direction: row;
flex-column Sets the flex direction to column (vertical) flex-direction: column;
flex-{breakpoint}-{row|column} Responsive flex direction (e.g., flex-md-row) Media query with flex-direction
justify-content-{value} Aligns items horizontally (start, end, center, between, around, evenly) justify-content: {value};
align-items-{value} Aligns items vertically (start, end, center, baseline, stretch) align-items: {value};
