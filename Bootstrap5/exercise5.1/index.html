Bootstrap Grid Alignment

This example demonstrates how to use justify-content-center and align-items-center to center content inside a row.

1. Horizontal Alignment with justify-content-*

The justify-content-* classes align items horizontally within a row. They work with flexbox properties to position items along the main axis (horizontally in a row).

justify-content-start (Default)

<div class="row justify-content-end"> <div class="col-4">Column</div> </div>

Column
Column
<div class="row justify-content-between"> <div class="col-4">Column</div> <div class="col-4">Column</div> </div>

justify-content-around

Column

Column

2. Vertical Alignment with align-items-*

The align-items-* classes align items vertically within a row. They work with flexbox properties to position items along the cross axis (vertically in a row).

Column
Column

3. Individual Column Alignment with align-self-*

The align-self-* classes allow you to align individual columns differently within the same row.

align-self-start
align-self-center
align-self-end
<div class="row"> <div class="col-4 align-self-start">align-self-start</div> <div class="col-4 align-self-center">align-self-center</div> <div class="col-4 align-self-end">align-self-end</div> </div>

4. Centering Content Both Horizontally and Vertically

Combine justify-content-center and align-items-center to center content both horizontally and vertically.

Centered Column
<div class="row justify-content-center align-items-center"> <div class="col-4">Centered Column</div> </div>
Perfect Centering

This technique is perfect for creating centered elements like modals, login forms, or hero sections.
