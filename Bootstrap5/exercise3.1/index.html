Bootstrap Responsive Grid Layout

This example demonstrates a responsive grid layout with columns that stack on mobile, display two per row on tablets, and three per row on desktops.

How the Grid System Works

Class Breakdown:
  • col-12: Takes full width (12 columns) on extra small devices (phones, <576px)
  • col-sm-6: Takes half width (6 columns) on small devices (tablets, ≥576px)
  • col-lg-4: Takes one-third width (4 columns) on large devices (desktops, ≥992px)
Bootstrap Breakpoints:
  • Extra small: <576px (Default, no prefix needed)
  • Small: ≥576px (sm)
  • Medium: ≥768px (md)
  • Large: ≥992px (lg)
  • Extra large: ≥1200px (xl)
  • Extra extra large: ≥1400px (xxl)
Tip: Resize your browser window to see how the columns respond to different screen sizes!
