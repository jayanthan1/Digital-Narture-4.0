Bootstrap Responsive Navbar - Exercise 10.1

This exercise demonstrates how to create a responsive navbar with Bootstrap 5, including a logo, navigation links, dropdown menu, and a search form.

Try resizing your browser window to see how the navbar responds to different screen sizes.

Navbar Features

Responsive Design

The navbar collapses into a hamburger menu on mobile devices and smaller screens.

Dropdown Menu

Includes a dropdown menu for organizing related navigation items.

Search Form

Integrated search form that adapts to different screen sizes.

Brand Logo

Includes a brand logo that maintains visibility across all screen sizes.

Implementation Code

Basic Navbar Structure

<nav class="navbar navbar-expand-lg navbar-dark bg-primary">
    <div class="container">
        <!-- Logo/Brand -->
        <a class="navbar-brand" href="#">
            <img src="logo.png" alt="Logo">
        </a>
        
        <!-- Mobile Toggle Button -->
        <button class="navbar-toggler" type="button" 
                data-bs-toggle="collapse" data-bs-target="#navbarMain">
            <span class="navbar-toggler-icon"></span>
        </button>
        
        <!-- Navbar Content -->
        <div class="collapse navbar-collapse" id="navbarMain">
            <!-- Navigation Links -->
            <ul class="navbar-nav me-auto mb-2 mb-lg-0">
                <li class="nav-item">
                    <a class="nav-link active" href="#">Home</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link" href="#">Features</a>
                </li>
                <!-- More nav items... -->
            </ul>
            
            <!-- Search Form -->
            <form class="d-flex">
                <input class="form-control me-2" type="search" placeholder="Search">
                <button class="btn btn-outline-light" type="submit">Search</button>
            </form>
        </div>
    </div>
</nav>
