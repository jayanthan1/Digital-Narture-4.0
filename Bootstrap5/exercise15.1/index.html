Bootstrap Border Utilities with Images

About This Exercise

This exercise demonstrates how to use Bootstrap's border utilities to enhance images. You'll see examples of:

  • Basic borders with border
  • Contextual border colors like border-primary
  • Border width variations with border-3
  • Rounded corners with rounded-circle and other utilities

Each example includes the image with the applied utilities and the code used to create it.

Basic Borders

Add a border to all sides of an image using the border utility class.

class="border"
Default border (1px solid #dee2e6)
class="border-top"
Border on top only
class="border-end"
Border on right side only
class="border-bottom"
Border on bottom only
class="border-start"
Border on left side only
class="border-top border-bottom"
Border on top and bottom
class="border-start border-end"
Border on left and right
class="border-0"
No border (removes borders)

Contextual Border Colors

Apply different colors to borders using contextual classes like border-primary, border-success, etc.

class="border border-primary"
Primary color border
class="border border-secondary"
Secondary color border
class="border border-success"
Success color border
class="border border-danger"
Danger color border
class="border border-warning"
Warning color border
class="border border-info"
Info color border
class="border border-light"
Light color border
class="border border-dark"
Dark color border

Border Width

Control the thickness of borders using border-1 through border-5 utilities.

class="border border-1 border-primary"
1px border width
class="border border-3 border-success"
3px border width
class="border border-5 border-danger"
5px border width

Rounded Corners

Apply rounded corners to images using utilities like rounded, rounded-circle, etc.

class="rounded"
Default rounded corners (0.25rem)
class="rounded-circle"
Circular shape (50% border-radius)
class="rounded-pill"
Pill shape (50rem border-radius)
class="rounded-0"
No rounded corners
class="rounded-top"
Rounded top corners only
class="rounded-end"
Rounded right corners only
class="rounded-bottom"
Rounded bottom corners only
class="rounded-start"
Rounded left corners only

Combining Border Utilities

Combine multiple border utilities to create more complex styles.

class="border border-3 border-primary rounded-circle"
Thick primary border with circular shape
class="border border-4 border-success rounded-3"
Thick success border with large rounded corners
class="border-top border-5 border-danger rounded-bottom"
Thick danger border on top with rounded bottom corners
