Icon-Only Buttons with Bootstrap Icons

Exercise 17.2 - Replacing text buttons with icon-only buttons

About This Exercise

This exercise demonstrates how to create icon-only buttons using Bootstrap Icons. We'll explore different button styles, sizes, and best practices for accessibility.

Key points covered: button variants, sizes, tooltips, and accessibility considerations.

Basic Icon Buttons

Replace text with icons while maintaining button functionality and accessibility:

Original Text Buttons:
<button class="btn btn-primary">Save</button>
<button class="btn btn-danger">Delete</button>
<button class="btn btn-success">Edit</button>
Icon-Only Buttons with Tooltips:
<button class="btn btn-primary btn-icon" data-bs-toggle="tooltip" title="Save">
    <i class="bi bi-save"></i>
    <span class="visually-hidden">Save</span>
</button>

Button Sizes

Icon buttons in different sizes:

<button class="btn btn-primary btn-icon btn-sm">...</button>
<button class="btn btn-primary btn-icon">...</button>
<button class="btn btn-primary btn-icon btn-lg">...</button>

Circular Buttons

Icon buttons with rounded corners:

<button class="btn btn-primary btn-icon btn-floating">
    <i class="bi bi-share"></i>
    <span class="visually-hidden">Share</span>
</button>

Outline Buttons

Icon buttons with outline style:

<button class="btn btn-outline-primary btn-icon">
    <i class="bi bi-download"></i>
    <span class="visually-hidden">Download</span>
</button>

Button Groups

Icon buttons grouped together:

<div class="btn-group" role="group" aria-label="Basic example">
    <button class="btn btn-primary">
        <i class="bi bi-chevron-left"></i>
        <span class="visually-hidden">Previous</span>
    </button>
    ...
</div>

Floating Action Buttons

Fixed position icon buttons:

Look at the bottom right corner of the page for floating action buttons.

.action-buttons {
    position: fixed;
    bottom: 2rem;
    right: 2rem;
    display: flex;
    flex-direction: column;
    gap: 0.5rem;
}

Best Practices

1. Accessibility
  • Always include visually hidden text for screen readers
  • Use tooltips to show button purpose on hover
  • Ensure adequate touch target size (at least 44x44px)
  • Maintain good color contrast
2. Visual Design
  • Keep icon styles consistent
  • Use appropriate icon sizes
  • Provide visual feedback on interaction
  • Consider using tooltips for clarity
