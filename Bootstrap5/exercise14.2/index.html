Responsive Sidebar Example

This example demonstrates a responsive sidebar that is:

  • Hidden on mobile devices (xs, sm breakpoints - below 768px)
  • Visible on tablets and desktops (md, lg, xl, xxl breakpoints - 768px and above)

On mobile devices, the sidebar is replaced with a dropdown menu in the header.

Resize your browser window to see how the layout changes. Look for the colored badge in the top-right corner to identify your current viewport size.

Implementation Details
Display Classes Used:
  • d-none d-md-block - Hide on mobile, show on tablet and above
  • d-block d-md-none - Show on mobile, hide on tablet and above
  • col-12 col-md-9 col-lg-10 - Adjust content width based on sidebar visibility
Key Components:
  1. Sidebar with navigation links
  2. Mobile dropdown menu alternative
  3. Responsive content area that adjusts width
Benefits of This Approach
  • Optimized for Mobile: On small screens, the sidebar is hidden to maximize content space
  • Better Navigation on Larger Screens: Persistent sidebar on tablets and desktops for easy navigation
  • Consistent Access to Navigation: Mobile users can still access all navigation options via the dropdown
  • Clean Implementation: Uses Bootstrap's built-in display utilities without custom JavaScript
